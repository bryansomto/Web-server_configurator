About

This web application is designed to ease the configuration of web servers for both beginners and expert developers. It is a project embarked upon through the ALX SE program. The web application is developed to make it easy for software engineering students working on web servers to configure an Nginx web server in a less difficult and visually interesting way. The Web Server Configurator provides a visual aided alternative to configure a remote Nginx web server. Configuration replicates package update and upgrade on linux machines, Nginx installation and configures UFW port for http/https.
